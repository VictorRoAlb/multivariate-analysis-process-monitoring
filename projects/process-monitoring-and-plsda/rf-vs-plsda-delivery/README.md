# Random Forest and PLS-DA for burnout-risk classification

Clean delivery package focused only on the two supervised models that were most useful for explaining the class-separation problem in the multivariate monitoring project:

1. `Random Forest`
2. `PLS-DA`

I reorganized this material from the original multivariate-monitoring coursework so the final submission is easier to read, run and defend.

## What is included

- `data/`
  Synthetic burnout-risk dataset used for the clean Python workflow.
- `src/build_rf_vs_plsda_delivery.py`
  Main script that trains both models, evaluates them and generates the final figures.
- `figures/`
  Exported comparison plots for metrics, confusion matrices and variable importance.
- `results/`
  CSV summaries and a short text summary with the final model outputs.
- `notebooks/`
  Space reserved for the simplified walkthrough notebook.

## Modelling focus

Both models are trained on the same six core psychological variables:

- `Anxiety_Score`
- `Mood_Stability_Score`
- `Sleep_Quality_Score`
- `Motivation_Level`
- `Emotional_Fatigue_Score`
- `Wellbeing_Index`

This keeps the comparison concise and directly interpretable.

## Output generated by the script

Running the script creates:

- `figures/rf_vs_plsda_metrics.png`
- `figures/rf_vs_plsda_confusion_matrices.png`
- `figures/rf_vs_plsda_importance.png`
- `results/rf_vs_plsda_metrics.csv`
- `results/plsda_cv_selection.csv`
- `results/rf_vs_plsda_feature_scores.csv`
- `results/summary.txt`

## How to run

```bash
python src/build_rf_vs_plsda_delivery.py
```
