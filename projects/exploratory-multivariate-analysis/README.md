# Exploratory Multivariate Analysis

Representative public material from the coursework folder `Analisis multivariante`.

## Technical focus

The original project folder covers a wider introductory multivariate-analysis block, including:

- PCA-style exploratory interpretation;
- factor-analysis context;
- discriminant-analysis context;
- cluster-analysis context;
- multiple correspondence analysis.

## Public material

- `notebooks/MCA.ipynb`
- `notebooks/Practica_1.ipynb`

These notebooks are kept as representative technical artifacts rather than as a full reproducible package. The associated spreadsheets and coursework submission files are intentionally excluded from the public export.

## Notes

- The source folder contains many PDF, DOCX and PPTX files that are not useful as GitHub portfolio assets.
- The repository therefore prioritizes technical notebooks and clean documentation over course-submission archives.
